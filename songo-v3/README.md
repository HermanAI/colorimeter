# Songo 237 - Version 3 (Modernisée)

## 📋 Description
Songo 237 est un jeu de semailles camerounais traditionnel en mode multijoueur Ajax avec support en temps réel et respect strict des règles de jeu.

## ✨ Améliorations V3
- ✅ Respect des règles clockwise (sens des aiguilles d'une montre)
- ✅ Correction des erreurs réseau (gestion CORS robuste)
- ✅ Interface moderne et réactive
- ✅ Polling amélioré avec détection de changements
- ✅ Validation stricte des coups
- ✅ Logging complet des actions

## 🚀 Installation
1. Extraire les fichiers dans htdocs/
2. Importer sql/schema.sql dans MySQL
3. Accéder à http://localhost/songo-v3/
